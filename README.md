# 🐠 IoT-Based Smart Fish Tank Monitoring & Automation System

An IoT-powered smart aquarium system that automates fish feeding, monitors water quality, and enables remote control through a Telegram Bot. This project was developed as a **Year 1 Semester 1 Group Project** with the goal of making aquarium maintenance easier, reliable, and efficient.

## 📌 Project Overview

Maintaining an aquarium requires regular attention such as feeding fish on time, monitoring water conditions, and changing water when necessary. For aquarium owners with busy schedules or frequent travel, these tasks can become challenging.

To solve this problem, we developed a smart aquarium system using **ESP32 and IoT technologies** that allows users to remotely monitor and control essential aquarium operations through a Telegram Bot.

The system supports both **small home aquariums and larger aquarium setups** by allowing users to configure tank-related settings.

---

# ✨ Features

## 🐟 Automated Fish Feeding
- Supports automatic fish feeding based on scheduled times.
- Users can configure up to **3 feeding times per day**.
- Manual feeding can also be triggered through the Telegram Bot.

## 📱 Telegram Bot Control
Users can remotely interact with the aquarium system through Telegram.

Available controls include:
- Feed fish manually
- Check water quality status
- Monitor water level
- Start water replacement
- Configure tank settings
- View system status

## 🌊 Automated Water Replacement
- Users can configure the aquarium tank size.
- Users can define how much water should be drained.
- The system ensures that enough water remains in the tank for fish safety.
- Water replacement is controlled using automated pumps.

## 💧 Water Quality Monitoring
- Uses a **Turbidity Sensor** to measure water clarity.
- Provides information about the current water condition.

## 📏 Water Level Monitoring
- Uses an **Ultrasonic Sensor** to measure water level.
- Helps control water replacement operations accurately.
